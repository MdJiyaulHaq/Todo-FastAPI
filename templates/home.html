Todo Application

Welcome to the Todo Application. Here you can manage your todos.

Endpoints

  • GET /todo/ - Retrieve all todos for the authenticated user.
  • GET /todo/{id} - Retrieve a specific todo by ID.
  • POST /todo/ - Create a new todo.
  • PUT /todo/{id} - Update an existing todo by ID.
  • DELETE /todo/{id} - Delete a todo by ID.

Authentication

This application requires authentication to access the endpoints. Please ensure you are logged in.
